Прога затирает все сейвы, кроме автосейвов в папке с DOOM 64 по пути C:\DOOM 64 v2.5
Таким образом, вы не можете сохраниться в любом месте в любое время.

Чтобы скомпилить программу, выполняем команду в PyCharm:

pip -m pip install --upgrade pip или easy_install -U pip
а затем
pyinstaller --onefile Doom64_SaveLoad_Block.py

Если надо, чтобы прога затирала вообще все сейвы, включая автосейвы,
убираем из исходника строчку and not file.startswith("auto") до двоеточия.

Проверка на удаление сейва: 2 секунды.

Программа весит ~ 6,5 mb.

Если хотите, чтобы прога, затирающая сейвы запускалась сразу с Думом,
можно сообразить простенький .bat-ничек, типа такого:

@echo off
start /B Doom64_SaveLoad_Block.exe
start /B ZDL.exe

И просто кинуть его в папку с Doom64.

!!! Программа будет затирать сейвы только при условии, что DOOM 64 лежит 
вот по такому пути C:\DOOM 64 v2.5 !!!
