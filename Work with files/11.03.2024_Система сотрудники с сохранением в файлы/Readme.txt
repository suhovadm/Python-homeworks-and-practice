Что это такое и что оно умеет?

Простая программа для работы с базой данных в текстовом формате.
Что умеет, по пунктам:

- делать бэкап БД в отдельный файл BACKUP.txt

- открывать БД прямо в шелле

- искать данные в БД по любому из параметров: ФИО | Возраст | Телефон | Номер кабинета

- добавлять данные в БД по шаблону: ФИО | Возраст | Телефон | Номер кабинета

- удалять данные из БД по ФИО.
* 22.06.2025 добавлена проверка на наличие сотрудника в БД.
* Если сотрудника в БД нет, выведется сообщение об ошибке.

- Заменять все параметры в БД.
* Пока только все параметры, в будущем функционал будет допилен.

- Выходить с автосейвом в отдельный файл OUT.txt на тот случай, если напортачили в основной базе.

- Кодировка программы по дефолту выставлена на UTF8, так что все файлы без проблем можно
редактировать в обычном Виндовом блокноте, или Notepad++
--------------------------------------------------------------------------------------------------
Исходники программы разделены на 8 файлов:
main-овый - главный и 7 файлов с основным функционалом.
+
Есть готовый билд, скомпиленный в .exe-шник и 3 файла с заготовками базы - IN.txt, OUT.txt и BACKUP.txt.
* Если удалить файл IN.txt, то при наполнении новой базы (п. 3 "Добавить") он создаст его сам.
* Также создадутся и файлы BACKUP.txt и OUT.txt при ручном и автоматическом сохранении.
