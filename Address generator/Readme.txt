Генератор внешних, внутренних (серых) IP v4 адресов, IP v6 адресов и MAC-адресов.

1) main.py - файл с основным кодом программы.
2) menu.py - функция с главным меню.
3) address_generator.py - функции для генерации IP v4, IP v6 и MAC-адресов.

Также в наличии есть готовая программа, собранная в .exe-шник.
