1.1 Усовершенствованная сортировка пузырьком ( bubble sort ).
2. Сортировка вставками ( insertion sort ).
3. Сортировка слиянием ( merge ).

------------------------------------------------------------------------------------------------------------------------
Самый быстрый и наиболее эффективный метод для общего случая: Timsort.
Самый быстрый в среднем случае: Быстрая сортировка методом Хоара (при правильной реализации и выборе опорного элемента).

Если нам важна именно максимальная эффективность в реальных условиях, то лучше выбрать Timsort.
Если нужен быстрый и простой алгоритм для случайных данных — подойдет быстрая сортировка методом Хоара.
