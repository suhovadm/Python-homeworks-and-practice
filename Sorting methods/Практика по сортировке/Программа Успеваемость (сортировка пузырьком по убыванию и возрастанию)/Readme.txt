Аналогичная версия программы "Успеваемость".
Данная версия реализована с помощью сортировки пузырьком в обе стороны.
У данной версии также есть главное меню и перехват ошибок.
Программа разбита на 7 файлов для более удобного и читабельного кода.
Каждый файл не превышает 100 строчек, включая пробелы и комментарии.
